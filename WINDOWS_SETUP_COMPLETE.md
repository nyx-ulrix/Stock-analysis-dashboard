### Option 3: Manual Setup

1. **Backend**:

    ```cmd

    python -m venv venv
    call venv\Scripts\activate.bat
    pip install -r requirements.txt
    python app.py
    ```

2. **Frontend** (in new terminal):
    ```cmd
    npm install
    npm run dev
    ```
